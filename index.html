How it works

Adding tasks
Add a task
Type in the "What needs doing?" box and press Enter or the ↑ button. Set priority, date, repeat, and tags in the options row before adding.
Priority
Click the Priority pill to cycle None → Low → Medium → High. The coloured stripe on the left of each task shows its priority. Change it anytime from the detail panel.
Due dates
Pick a date when adding, or edit it in the detail panel. Overdue tasks show red, today shows amber.
Recurring tasks
Click Repeat to cycle Daily, Weekdays, Weekly, or Monthly. Completing a recurring task auto-reschedules it to the next occurrence.
Tags
Click a tag pill to attach it. Click "+ tag" to create a new one. Filter by clicking a tag chip below the filter tabs. Hover a chip to rename or delete it.
Managing tasks
Complete a task
Click the circle to mark done. Click again to unmark. Recurring tasks reschedule automatically.
Pin tasks
Hover a task and click the pin icon to pin it. Pinned tasks always appear at the top of the list.
Edit task text
Hover a task and click the pencil icon. Press Enter to save or Esc to cancel.
Task details
Click the ↓ chevron to expand a task. Inside: priority, due date, recurrence, time estimate, notes, and subtasks.
Drag to reorder
Hover to reveal the grip handle on the left, then drag up or down.
Delete & undo
Hover and click the trash icon. An Undo toast appears for 5 seconds at the bottom.
Organising
Search
Filters tasks in real time by name, note, or tag. Press / to jump to it, Esc to clear.
Bulk actions
Click "Select" to enter selection mode, tick tasks, then Complete or Delete all at once.
Auto-backup
Click "Link file" in the backup bar to pick a local JSON file. Every change is saved to it automatically. Click "Restore" to load tasks from a backup file.
Statistics
Click the chart icon (or press S) to see your completion stats, streaks, and task breakdowns.
Confetti
Complete every task on your list and watch what happens 🎉
Keyboard shortcuts
Shortcuts
N — focus the new task input / — focus search S — open statistics D — toggle dark mode ? — open this help page Enter — add task / save edit Esc — go back / cancel / clear search
